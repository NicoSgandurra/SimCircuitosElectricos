# 🔬 Simuladores de Ingeniería Eléctrica y Térmica

Este repositorio contiene una colección de simuladores interactivos para materias de ingeniería, desarrollados con tecnologías web estándar (HTML, CSS y JavaScript). El objetivo es facilitar la comprensión de conceptos teóricos mediante herramientas visuales, prácticas y libres de publicidad.

## 🚀 Acceso Online

El proyecto está disponible online y se puede acceder a través del siguiente enlace:

🔗 **[https://nicosgandurra.github.io/SimCircuitosElectricos/](https://nicosgandurra.github.io/SimCircuitosElectricos/)**

Al ingresar, verás un **Hub Central** desde donde podrás navegar a cualquiera de los simuladores disponibles.

## 📚 Simuladores Disponibles

A continuación se detalla la funcionalidad de cada simulador, organizado por materia:

### ⚡ Circuitos Eléctricos I

1.  **Sistemas de Ecuaciones Lineales**
    *   **Archivo**: `Circuitos Eléctricos I/SistEcuacionesLineales.html`
    *   **Función**: Resuelve sistemas de ecuaciones lineales complejas utilizando matrices y el método de Cramer.
    *   **Uso**: Ideal para verificar cálculos de mallas y nodos en circuitos. Permite ingresar coeficientes (reales o complejos) y obtener las incógnitas.

2.  **Transformación Estrella - Triángulo**
    *   **Archivo**: `Circuitos Eléctricos I/Estrella-Triangulo.html`
    *   **Función**: Realiza conversiones entre configuraciones de resistencias en Estrella (Y) y Triángulo (Δ).
    *   **Uso**: Ingresa los valores de las resistencias en una configuración para obtener sus equivalentes en la otra. Útil para simplificar redes resistivas.

3.  **Resultados de Trabajos Prácticos**
    *   **Archivo**: `Circuitos Eléctricos I/ResultadosEjercicios.html`
    *   **Función**: Verificación de respuestas para guías de ejercicios.
    *   **Uso**: Los alumnos pueden ingresar sus resultados numéricos y el sistema valida si son correctos, proporcionando una calificación basada en la efectividad.

### 🔋 Circuitos Eléctricos II

4.  **Factor de Potencia**
    *   **Archivo**: `Circuitos Eléctricos II/FactorDePotencia.html`
    *   **Función**: Análisis vectorial de potencias (Activa, Reactiva, Aparente) y corrección del factor de potencia.
    *   **Uso**: Simula cargas inductivas/capacitivas y calcula el banco de capacitores necesario para corregir el factor de potencia a un valor deseado.

5.  **Neutro Flotante**
    *   **Archivo**: `Circuitos Eléctricos II/NeutroFlotante.html`
    *   **Función**: Simulación de sistemas trifásicos desequilibrados sin conexión a tierra (neutro flotante).
    *   **Uso**: Visualiza el desplazamiento del punto neutro y las tensiones resultantes en las cargas ante desequilibrios.

6.  **Resultados de Trabajos Prácticos**
    *   **Archivo**: `Circuitos Eléctricos II/ResultadosEjercicios.html`
    *   **Función**: Similar al de Circuitos I, permite la autoevaluación de ejercicios de la materia.

### ⚙️ Máquinas Eléctricas

7.  **Motores Paso a Paso**
    *   **Archivo**: `Máquinas Eléctricas/MotoresPaP-Rev4.html`
    *   **Función**: Control y visualización de motores paso a paso.
    *   **Uso**: Permite seleccionar tipos de secuencia (wave, full-step, half-step) tanto para motores unipolares como bipolares, visualizando la activación de bobinas y el movimiento del rotor.

### 🔥 Máquinas Térmicas

8.  **Diagrama de Oswald**
    *   **Archivo**: `Máquinas Térmicas/oswald.html`
    *   **Función**: Cálculo de parámetros de combustión.
    *   **Uso**: Genera el diagrama de Oswald para analizar la combustión completa e incompleta basándose en la composición del combustible y los gases de escape.

## 🛠️ Cómo Usar Localmente

Si deseas ejecutar estos simuladores en tu propia computadora sin conexión a internet:

1.  **Clona o descarga** este repositorio.
2.  Busca el archivo **`index.html`** en la carpeta raíz.
3.  Ábrelo con cualquier navegador web moderno (Firefox, Zen, Chrome, Brave, Edge, etc.).
4.  Navega a través del menú principal tal como lo harías en la versión online.

## 💻 Tecnologías

*   **HTML5**: Estructura semántica.
*   **CSS3**: Estilos, diseño responsive y temas (Claro/Oscuro).
*   **JavaScript (Vanilla)**: Lógica de simulación y cálculo matemático.
